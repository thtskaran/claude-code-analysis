# Security Analysis

Security hardening and threat analysis of Claude Code. Includes the 4,437-line bash parser (fail-closed design), de-sanitization lookup tables, injection vulnerability assessment, and comprehensive security audit.

## Files

- **bash-parser-analysis.md** - Complete breakdown of the 4,437-line bash parser and command validation logic
- **desanitization-table.md** - Lookup tables and patterns for detecting de-sanitization vulnerabilities
- **injection-vulnerability-assessment.md** - Analysis of prompt injection, command injection, and other attack vectors
- **security-audit.md** - Full security audit covering cryptography, authentication, and access controls
- **threat-model.md** - Threat modeling and attack surface documentation
