# Tools, Skills, and Plugins

Complete inventory of Claude Code's tool ecosystem, bundled skills, and plugin architecture. Covers tool contracts, command surface, skill catalog, and plugin marketplace mechanisms.

## Files

- **tool-contracts.md** - Specification and interface definitions for all integrated tools
- **command-surface.md** - Complete CLI and programmatic command interface documentation
- **bundled-skills-catalog.md** - Full inventory of shipped skills with examples and usage patterns
- **plugin-architecture.md** - Plugin discovery, loading, isolation, and marketplace mechanisms
- **mcp-integration.md** - Model Context Protocol integration and MCP server specifications
