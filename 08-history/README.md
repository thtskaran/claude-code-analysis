# History and Attribution

Product evolution and contribution tracking. Documents the migration from earlier versions of Claude Code, architectural transitions, and an attribution matrix identifying contributions by team members across all subsystems.

## Files

- **migration-history.md** - Version history, breaking changes, and architectural transitions
- **contribution-matrix.md** - Attribution and contribution tracking by module and team member
- **release-timeline.md** - Major releases, feature additions, and deprecation schedule
- **design-decisions.md** - Rationale and historical context for key architectural choices
