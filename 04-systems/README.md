# Systems, Configuration, and Telemetry

Complete inventory of runtime configuration, feature flags, environmental settings, and telemetry instrumentation. Covers 90+ feature flags, all constants and thresholds, environment variables, settings schemas, and telemetry event definitions.

## Files

- **feature-flags.md** - All 90+ feature flags with descriptions and rollout status
- **configuration-schema.md** - Complete settings schema, defaults, and validation rules
- **environment-variables.md** - All env var requirements and runtime configurations
- **telemetry-events.md** - Event definitions, sampling rates, and instrumentation points
- **constants-and-thresholds.md** - All hardcoded constants, rate limits, and performance thresholds
