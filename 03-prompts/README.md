# Prompt Engineering

Complete collection of system prompts and behavioral instructions used by Claude Code. The prompt-bible.md contains the full archive of 2.1MB and 32,000+ lines of verbatim prompts extracted from the codebase.

## Files

- **prompt-bible.md** - Complete 32K-line collection of all system prompts, user-facing instructions, and behavioral directives
- **model-behavioral-contract.md** - How Claude Code instructs Claude models to behave, safety rules, and capability boundaries
- **prompt-patterns.md** - Common patterns and templates used across the codebase
- **system-instructions.md** - Core system-level instructions and initialization prompts
